If you have selected a repository, it will be shown on the map as a yellow dot, and the ecosystem hierarchy it has been classified into will be outlined in yellow circles. Projects that have been classified as related will be indicated with purple dots, and the ecosystems for those related projects will be outlined in purple circles.

The clustering for this site is based on affinity propagation. For this app, repo A has an affinity for repo B if enough of repo A's contributors also contributed to, or starred, repo B. If repo A has an affinity for repo B, and repo B has an affinity for repo A, the two repos are said to be related. For more information, click here.
